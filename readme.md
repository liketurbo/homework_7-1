**Time and space complexity**
  * Native - `T: n/a` `S: n/a`
  * Brute Force - `T: O(n*m*k)` `S: O(n)`
  * Rabin-Karp - `T: O(n*(m+k)), O(n*(m*k))` `S: O(n)`
  * KMP - `T: O(n*(m+k)` `S: O(n+k)`
  * Suffix Tree - `in development`
